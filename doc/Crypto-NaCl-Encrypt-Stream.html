Given a Nonce n, plaintext p and SecretKey sk, encrypt n p sk encrypts the message p using SecretKey sk and returns the result.

encrypt guarantees the resulting ciphertext is the plaintext bitwise XOR'd with the result of streamGen. As a result, encrypt can also be used to decrypt messages.
